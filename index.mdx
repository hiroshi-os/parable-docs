Welcome to **ted** — a minimalist desktop code editor designed from the ground up for AI-assisted development. Whether you're running autonomous AI agents across your codebase, reviewing git history, or writing code with LSP-powered completions, ted keeps everything in one fast, native application. ted is built for developers who want a focused, agent-first environment without the overhead of a full-featured IDE.

## What's in ted

ted ships with a focused set of capabilities designed to cover the full AI-assisted development workflow — from editing and running agents to reviewing history and customizing your environment.

<CardGroup cols={3}>
  <Card title="Editor" icon="code">
    CodeMirror 6 engine with tree-sitter syntax highlighting, multi-pane layouts, tabs, and LSP-powered completions and diagnostics.
  </Card>
  <Card title="AI Agents" icon="robot">
    An LLM agent loop that supports Ollama, OpenAI, Anthropic, and Google. Run single agents or coordinate multiple agents with Swarms.
  </Card>
  <Card title="Git" icon="code-branch">
    Integrated source control with visual commit history, file blame, staging area, and commit — no terminal required.
  </Card>
  <Card title="Terminal" icon="terminal">
    A built-in PTY terminal lets you run commands, scripts, and dev servers without switching to an external app.
  </Card>
  <Card title="Extensions" icon="puzzle-piece">
    Build and load custom extensions using the TedAPI. Add new panels, commands, keybindings, and tool integrations.
  </Card>
  <Card title="Configuration" icon="gear">
    Semantic theming and a settings interface let you tailor ted's appearance and behavior to your workflow.
  </Card>
</CardGroup>
